Huffman

Веб-приложение для сжатия текста с использованием алгоритма Хаффмана на чистом JavaScript.
Проект работает на чистом JavaScript без сборок и зависимостей.

Запуск:

Live Server (рекомендуется)
1. Установите расширение **Live Server** в VS Code
2. Откройте проект в VS Code
3. Правой кнопкой по файлу `main.html` → **"Open with Live Server"**
4. Проект откроется в браузере по адресу `http://localhost...`
